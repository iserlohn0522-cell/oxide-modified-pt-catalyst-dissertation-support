# Zr-JC-star

This package contains the selected retained and displaced endpoint inputs, coordinates, declared CP2K evidence excerpts, and accepted energies for **Zr-JC-star**. The reported within-pair difference is 1.360508 eV.

Protocol: `M1_600Ry_nonperiodic`. Alternate displaced support basin from the same retained endpoint and displacement.

Geometry lineage: The retained endpoint is shared with Zr-JC; the displaced endpoint is an alternate Pt6-fixed support-relaxation basin.

Each `source_sha256.unpublished_full_output` value identifies the accepted full CP2K output used to create the public excerpt. The full output and scheduler records are intentionally not distributed.
