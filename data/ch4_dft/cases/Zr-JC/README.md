# Zr-JC

This package contains the selected retained and displaced endpoint inputs, coordinates, declared CP2K evidence excerpts, and accepted energies for **Zr-JC**. The reported within-pair difference is 2.946157 eV.

Protocol: `M1_600Ry_nonperiodic`. Primary selected junction-contact endpoint.

Geometry lineage: Both endpoint geometries derive from Pt6-fixed support relaxation; final energies are fixed-geometry single points.

Each `source_sha256.unpublished_full_output` value identifies the accepted full CP2K output used to create the public excerpt. The full output and scheduler records are intentionally not distributed.
