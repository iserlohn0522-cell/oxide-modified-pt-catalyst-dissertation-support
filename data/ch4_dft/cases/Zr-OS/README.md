# Zr-OS

This package contains the selected retained and displaced endpoint inputs, coordinates, declared CP2K evidence excerpts, and accepted energies for **Zr-OS**. The reported within-pair difference is 2.112990 eV.

Protocol: `M2_400Ry_periodic`. Lineage-consistent control compared only within the m2 control protocol.

Geometry lineage: The retained geometry came from an earlier closed-shell unconstrained optimization; its reported energy is a later LSD fixed-geometry single point. Only the displaced-endpoint optimization fixed the six Pt atoms.

Each `source_sha256.unpublished_full_output` value identifies the accepted full CP2K output used to create the public excerpt. The full output and scheduler records are intentionally not distributed.
